\doxysubsection{Detailed Description}
The Caretaker class for the Memento pattern. It is responsible for keeping track of the {\ttfamily location\+Coordinates\+Memento} objects. This class can add new memento objects to the internal list and retrieve them based on an index, enabling the functionality to restore to a previously saved state.
